## PrimeLine Auto
PrimeLine Auto e uma SPA desenvolvida em Angular com Tailwind CSS para apresentar uma loja de pecas automotivas premium. O site possui pagina inicial, catalogo de produtos, promocoes, contato e carrinho de compras interativo.

## Como executar o projeto
O projeto Angular esta dentro da pasta principal.

## Acesse a pasta do projeto:
cd principal
## Instale as dependencias:
npm install
## Execute o servidor de desenvolvimento:
npm start
## Abra o navegador em:
http://localhost:4200/
